About This App

A simple and fast movie & TV show tool powered by the free IMDB API.

How It Works

  • Results are fetched in real-time from the imdbapi.dev API.
  • Only movies and TV shows are shown.

Free API Used

This app uses the 100% free and public API from:
https://imdbapi.dev/

No API key required • 1,000 requests/day • Perfect for learning and small projects.

Features

  • Posters, ratings, and release years
  • Built with Django 5.2 + Bootstrap 5.3.8
